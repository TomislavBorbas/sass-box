HTML opening Arhitecture

<div class="container"> <div class="container__inner"> </div> </div>

Rules

  • P is used for large text
  • Span is used for small texts
  • Label is used only for labels
  • Br is forbidden
  • Hr is forbidden
  • Flex is used only for parent and first child
  • Flex is forbidden on buttons
  • If ViewportHeight(vh) is used, fallback for mobile devices should be set(min-height)
  • If container is not full-width, container-size from containers(container.scss) should be used
  • Titles should be defined in base -> typography, classes should be used for titles(ex. h1 class="h1--title-primary")
  • Texts on background/image should be wrapped in class text-with-image(base -> typhography.scss)
  • All margins and paddings are defined with gutters(spacings) (variables -> gutter.scss)
